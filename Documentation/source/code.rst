Code
====

Data for Slovenian analysis is combined from various data sources with different file type and column layout:

#. SCADA sensor data:

   #. 'SCADA_data' folder contains original csv files stored in MOBISTYLE database (DB)

   #. 'SCADA_missing' folder contains csv files that are used to fill the missing data gaps

#. INAP sensor data: Web service of IRI-UL

#. MOBISTYLE Expert tool data: Platform to access MOBISTYLE DB monitoring data (all demonstration cases)
