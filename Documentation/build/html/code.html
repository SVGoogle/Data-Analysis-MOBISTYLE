Code¶

Data for Slovenian analysis is combined from various data sources with different file type and column layout:

  1. SCADA sensor data:
    1. ‘SCADA_data’ folder contains original csv files stored in MOBISTYLE database (DB)
    2. ‘SCADA_missing’ folder contains csv files that are used to fill the missing data gaps
  2. INAP sensor data: Web service of IRI-UL
  3. MOBISTYLE Expert tool data: Platform to access MOBISTYLE DB monitoring data (all demonstration cases)
