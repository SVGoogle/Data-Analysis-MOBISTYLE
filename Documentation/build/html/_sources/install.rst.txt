.. _install:

Installation
============

1. Run the following command in your command line:

.. code-block:: console   

   mkdir MOBISTYLE
   cd MOBISTYLE

2. In Git command line turn the MOBISTYLE directory into a git repository and clone the project repository to it:

.. code-block:: bash   

   $ git init
   $ git clone https://github.com/SVGoogle/Data-Analysis-MOBISTYLE.git


.. note:: If you are new to GitHub and git version control system follow the `git cheat sheet <https://github.github.com/training-kit/downloads/github-git-cheat-sheet.pdf>`_
